# PLAN

## 작업명
bareun.vercel.app 공개 화면/동작 기반 원본 구현용 명세 작성

## 목표
공개 사이트(https://bareun.vercel.app/)를 관찰하여, 소스코드 복사 없이 **사용자에게 보이는 화면 구조와 기능 동작**을 재구성할 수 있는 참조 명세를 문서화한다.

## 산출물
- `docs/reference-audit.md`

## 범위
- 포함: 화면 구성, 단계/흐름, 입력/버튼, 반응형 차이, 시각 스타일 톤, 인터랙션, 확실/추정/불확실 구분
- 제외: 실제 구현 코드 작성, 기능 개발, 리팩터링, 배포, 테스트 자동화

## 진행 절차
1. 대상 사이트 접속 및 공개 UI 텍스트/흐름 관찰
2. 단계형 화면(대여/반납/관리)의 주요 상태 전이 정리
3. 데스크톱/모바일 차이 및 레이아웃 규칙 요약
4. 확실/추정/불확실 항목을 분리해 명세화
5. 문서 검토 후 종료

## 완료 기준
- 요구된 항목이 모두 `docs/reference-audit.md`에 포함됨
- 구현 지시 없이도 화면/기능 구조를 재현 가능한 수준의 명세가 제공됨
- 추정과 불확실 영역이 명시적으로 분리됨
